# Workout Genie

A simple web app that generates workout routines based on user input.

## Features
- Dynamic workout suggestions
- Simple and clean UI
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript

## Live Demo
https://workout-geniee.vercel.app/

## How it works
Users input their preferences → app generates a suitable workout plan using predefined logic.
